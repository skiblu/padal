வாராரய்யா வாராரே வழிநடையா வாராரே\
வழிநடையாய் வாராரே பெருவழியில வாராரே \
பெருவழியில வாராரே பேரின்பம் தாராரே

அச்சன் கோவில் அரசனும் ஆடிப்பாடி வாராரே \
ஆரியங்காவு அய்யாவுமே ஆனந்தமாய் வாராரே \
குளத்துப்புழை பாலனுமே குறைகள் தீர்க்க வாராரே

எருமேலி சாஸ்தாவுமே ஏகாந்தமாய் வாராரே \
ஏகாந்தமாய் வாராரே எங்கள் தெய்வம் வாராரே \
பெரியான வட்டத்துல பெருமாளும் வாராரே \
சிறியான வட்டத்துல சிவனோடும் வாராரே

காளைகட்டி நாதனுமே வேலை தீர்க்க வாராரே \
அழுதையில குளிச்சிப்புட்டு ஹரிஹரனும் வாராரே\
ஹரிஹரனும் வாராரே ஐயப்பனும் வாராரே

கரிமலையின் ஏத்தத்திலே கண்ணன் மகன் வாராரே \
கண்ணன் மகன் வாராரே கருப்பனோட வாராரே

பம்பையில குளிச்சிப்புட்டு பாலகனும் வாராரே \
பாலகனும் வாராரே பந்தளந்தரசன் வாராரே

நீலிமலை ஏத்தத்திலே நிக்காமலே வாராரே \
நிக்காமலே வாராரே பிரம்மச்சாரி வாராரே

அப்பாச்சி மேடேறி ஹரிஹரனும் வாராரே \
ஹரிஹரனும் வாராரே ஐயப்பனும் வாராரே

சபரிமலை நாதனுமே சபலம் போக்க வாராரே \
சபலம் போக்க வாராரே சாஸ்தாவும் வாராரே

பதினெட்டு படியேறி சாஸ்தாவும் வாராரே \
வில்லெடுத்து அம்பெடுத்து வேட்டையாடி வாராரே \
வேட்டையாடி வாராரே விரமணிகண்டன் வாராரே

ஐயப்ப பஜனைக்குத்தான் ஆனந்தமாய் வாராரே \
வந்தவரை வாழ்த்திடவே கருணையோடு வாராரே

`வாராரய்யா வாராரே வழிநடையா வாராரே `(2)


`சுவாமியே... சரணம் ஐயப்பா!`
